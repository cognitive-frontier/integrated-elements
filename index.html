Classical Elements

Classical Elements occupy same-order cells: irrational aspects pair with irrational axes, and rational aspects pair with rational axes. Example: N × Judicious = Ne, N × Decisive = Ni, T × Merry = Ti, T × Serious = Te.

Integrated Elements

Integrated Elements occupy cross-order cells: irrational aspects pair with rational axes, and rational aspects pair with irrational axes. Example: N × Merry = N?, N × Serious = N!, T × Judicious = T?, T × Decisive = T!.

Use it Properly

Integrated Elements are resonance coordinates, not extra functions. Use them after the Classical stack is stable: first locate the Classical Ego, then use the Integrated pair to explain Mirage-like similarity, vibe, and cross-block resemblance.

Typing Rule: If the Integrated layer contradicts the Classical layer, trust the Classical stack first. The Integrated layer explains atmosphere; it does not replace the metabolic machinery.
